# Script folder
This folder should contain *as submodules* the bash scripts that start a particular feature/algorithm (e.g. SLAM). The folder structure should be the following:
```bash
scripts
├── <submodule_a>
│   ├── <algorithm_1>
│   ├── ...
│   └── <algorithm_n>
├── <submodule_b>
│   ├── <algorithm_1>
│   ├── ...
│   └── <algorithm_n>
└── README.md           # <-- This README.md
```
